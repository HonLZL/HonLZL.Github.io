第一章 动态规划

1 数字三角形

在矩阵里走路，从左上角走到右下角问题

传纸条

关键在于分析矛盾情况，这个点是否可以走，可以由哪里来。根据状态计算来到当前点的值。

通常设置 f(k,i,u) 为集合，k 为总共走了多少步，i 为第一个点的横坐标，u 为第二个点的横坐标。由于两个点走的步数相同，因此当 i==u 时，二者相遇到同一点。

基础版

for(int i=1;i<=n;i++) f[i]=1;  // memset 只能初始化为 0 -1 0x3f
for(int i=1;i<=n;i++)
        for(int j=1;j<i;j++)

1 上升下降最长子序列：登山

先从左往右求以 i 为结尾的最长上升子序列 f，再从右往左求以 i 为结尾的最长上升子序列 g，则答案为 max(f(i)+g(i)-1)。

4 数组中，最少的不上升子序列覆盖数组：拦截导弹

  • 第一问求一个拦截系统最多能拦截多少个导弹，求最长不上升子序列即可。

  • 第二问求至少需要多少个拦截系统能拦截所有导弹：贪心计算

    维护多个系统末尾高度数组，这个数组是递增的，当新来一个导弹，若导弹大于所有系统的末尾高度，则新开一个系统并将该导弹高度作为新系统末尾高度，否则寻找一个不小于导弹高度中最小的那个与之交换。

    ==调整法==贪心证明：假设第 i 个数 x ，贪心解和最优解放置方案不一样，

    x 在贪心解中接在 a 后面，在最优解中接在 b 后面，此时 $x\le a\le b$ ，那么必然可以将贪心解 x 后面的序列和最优解 x 后面的序列进行交换，因此在结果不变的情况下贪心解必然可以转换为最优解，因此贪心是正确的。

6 时间复杂度的优化 $O(nlogn)$ 

int stk[N],tt=0;  // 栈是递增的
stk[0]=-inf; // 栈底置为无穷小
for(int i=0;i<n;i++)
{
    if(a[i]>stk[tt])   // 直接加后面
    	stk[++tt]=a[i];  
    else   // 找最小的大于等于 a[i] 的替换掉，栈仍然单调
    {
        int p=lower_bound(stk,stk+tt,a[i])-stk;
        stk[p]=a[i];
    }
}
// tt 即为最大长度，stk 为选的数

7 最长上升公共子序列

寻找两个数组的最长公共子序列

f(i,j) 表示 $a(1\sim i)$，$b(1\sim j)$ 中以 $b_j$ 结尾的最长公共上升子序列的长度最大值。

  1. 不包含 $a_i$ ，最大值为 $f[i-1,j]$
  2. 包含 $a_i$，如果 $a_i\ne b_j$，相当于不包含 $a_i$。所以考虑 $a_i=b_j$， 然后继续划分，子序列是由 $b$ 前面哪个承接过来的。所以要枚举 b 数组前面的每一项，若 $b_k<b_j$，则可以转移过来，所以最大长度是 $f[i,k]+1$。
for(int i=1;i<=n;i++)
	for(int j=1;j<=n;j++)
    {
        f[i][j]=f[i-1][j];
        if(a[i]==b[j])
        {
            int ma=1;
            for(int k=1;k<j;k++)
                if(b[k]<a[i]) ma=max(ma,f[i,k]+1);
            f[i][j]=max(f[i][j],ma);
        }
    }
// 优化：由于是枚举小于 ai 或者说是 bj 的值，而第二层循环就是在已知ai的情况下枚举 b[1~j), 所以可以承担这个功能
for(int i=1;i<=n;i++)
{
    int ma=1;
    for(int j=1;j<=n;j++)
    {
        f[i][j]=f[i-1][j];
        if(a[i]==b[j])  f[i][j]=max(f[i][j],ma);
        if(b[j]<a[i])   ma=max(ma,f[i][j]+1);
    }
}

for(int i=1;i<=k;i++)

f[0]=1;  //根据实际意义初始化，从前 0 个选数，和为0的方案是1
for(int i=1;i<=n;i++)
    for(int j=m;j>=a[i];j--)
        f[j]=f[j]+f[j-a[i]];

4 单调队列优化的多重背包

多重背包问题 III

二进制优化时间复杂度 $O(NMlog(s))$，单调队列的优化 $O(NM)$。

题解

考虑用完全背包的方式取优化多重背包。将多重背包写成

给每个物品按它的体积 v 的余数分类，余数：[0,v)

f[0],f[v],f[2v],···,f[kv]

f[1],f[1+v],f[1+2v],···,f[1+kv]

······

f[j],f[j+v],f[j+2v],···,f[j+kv]

j 为 v 的余数。f[j+kv]是由前面不超过 s 的同类值传递得到的。

滑出窗口情况：因为只有s个物品，所以最多 -sv，超过 s 时要出队头，对应体积就是小于 k-sv 之外的要出队。

时间复杂度 nv(V/v)=nV，O(nV) 

for(int i=1;i<=n;i++)
{
    memcpy(g,f,sizeof(f));  // 滑动窗口建立在 f 的上一次结果中
    cin>>v>>w>>s;
    for(int j=0;j<v;j++)   // 拆分成 v 个类
    {
        int hh=0,tt=-1; // 队列清空
        for(int k=j;k<=V;k+=v)  
        {
            // 滑出窗口，体积小于全拿时的体积
            if(hh<=tt && q[hh]<k-s*v) hh++;
            // 用最大值(队头)更新 f，窗口在旧 f 滑动，因此备份 g
            if(hh<=tt) f[k]=max(g[k],g[q[hh]]+(k-q[hh])/v*w);  // 注
            // 当前值比队尾的物品个数更好，队尾元素出队
            // g[q[tt]]-(q[tt]-j)/v*w<=g[k]-(k-j)/v*w
            while(hh<=tt && g[k]>=g[q[tt]]+(k-q[tt])/v*w)  tt--;
            q[++tt]=k;   // 当前值入队
        }
    }
}
cout<<f[V]<<endl;

注：为什么选取最大可以放在while前面

f[i，j]=max(f[i-1，j], f[i-1，j-v]+w, f[i-1，j-2v]+2w… f[i-1，j-sv]+sw)

g[k]=f[i-1,k]，所以并不会漏掉 f[i-1,j] 项的。放在 q[++tt]=k 后面也没错。

5 混合背包问题，物品可以用 1 次，$s_i$ 次，无限次。

分别套用各自的模板

for(int i=1;i<=n;i++)
{
    memcpy(g,f,sizeof(f));
    cin>>v>>w>>s;
    if(s==-1)   // 01背包
        for(int j=V;j>=v;j--)
            f[j]=max(f[j],f[j-v]+w);
    else if(s==0)   // 完全背包
        for(int j=v;j<=V;j++)
            f[j]=max(f[j],f[j-v]+w);
    else   // 多重背包
        for(int j=0;j<v;j++)
        {
            int hh=0,tt=-1;
            for(int k=j;k<=V;k+=v)
            {
                if(hh<=tt && q[hh]<k-s*v) hh++;
                if(hh<=tt) f[k]=max(g[k],g[q[hh]]+(k-q[hh])/v*w);
                while(hh<=tt && g[k]-(k-j)/v*w>=g[q[tt]]-(q[tt]-j)/v*w) tt--;
                q[++tt]=k;
            }
        }
}
cout<<f[V]<<endl;

6 二维费用的背包问题

所选的物品含有两个条件，例如体积和重量，体积和价格。

例题：有 $N$ 件物品和一个容量是 $V$ 的背包，背包能承受的最大重量是 $M$。每件物品只能用一次。体积是 $v_i$，重量是 $m_i$，价值是 $w_i$ 。求解将哪些物品装入背包，可使物品总体积不超过背包容量，总重量不超过背包可承受的最大重量，且价值总和最大。

二维 f ，f[i,j]中 i 表示体积，j 表示价值。循环枚举两个费用即可。

for(int i=1;i<=N;i++)
{
    int v,m,w;  cin>>v>>m>>w;
    for(int j=V;j>=v;j--)
        for(int k=M;k>=m;k--)
            f[j][k]=max(f[j][k],f[j-v][k-m]+w);
}
cout<<f[V][M]<<endl;

7 费用最多、恰好、至少问题

费用最多是j：常见的背包都是此种情况，全部初始化为 0，不能枚举到 0

费用恰好是j：f[0]=0, 其他为正无穷，不能枚举到 0

费用至少是j：f[0]=0, 其他为正无穷，j 可以枚举到 0

潜水员 ：氧气至少是 n，氮气至少是 m，求最小重量

memset(f,0x3f,sizeof f);
f[0][0]=0;
for(int i=1;i<=k;i++)
{
    int a,b,w;  cin>>a>>b>>w;
    for(int j=n;j>=0;j--) // 可以枚举到 0，j-a<0状态是合法的
        for(int k=m;k>=0;k--)
            f[j][k]=min(f[j][k],f[max(0,j-a)][max(0,k-b)]+w);
}

9 背包问题的方案与方案数

求背包问题的具体方案

要求输出字典序最小，假设第一个物品在其中一个最优解中，那么该物品必选，然后问题转化为从 [2,n] 中寻找最优解。之前 f[i,j] 保存的是从前 i 个物品选择的最优解，所以在这题中应该表示从 [i,n] 中选择的最优解，所以要倒叙枚举每个物品递推式为：f[i,j]=max( f[i+1,j], f[i+1,j-v[i]]+w[i] )

在输出时，正着枚举物品；

  1. $f(i,m)=f(i+1,m-v[i])+w[i]\ \ 且\ \ f(i,m)\ne f(i+1,m)$ : 必须选 i 才能得到最优解
  2. $f(i,m)\ne f(i+1,m-v[i])+w[i] $ : 任何最优解都不包括 i
  3. $f(i,m)=f(i+1,m-v[i])+w[i]$ : 最优解可以有 i 也可以没有

01 背包求方案数，先用二维数组分析，再按 f 优化的套路优化 g

与求最大价值的顺序相同，只是属性变为数量

memset(f,-0x3f,sizeof f);
f[0]=0;  g[0]=1;
for(int i=1;i<=n;i++)
{
    int v,w;  cin>>v>>w;
    for(int j=m;j>=v;j--)
    {
        int ma=f[j], cnt=0;
        if(j>=v)     ma=max(ma,f[j-v]+w);
        if(ma==f[j]) cnt=g[j];  // 不选 i 可以到最优解
        if(j>=v&&ma==f[j-v]+w)    // 选 i 也可以到最优解
             cnt=cnt+g[j-v];
        g[j]=cnt%mod;   f[j]=ma;
        res=max(res,ma);
    }
}
for(int i=0;i<=m;i++)
    if(f[i]==res) ans=(ans+g[i])%mod;
cout<<ans<<endl;

如果定义为==不恰好==，那么 g[0,i] 都为 0，因为什么都不选对应一个方案。那么 g[i,j] 即为从前 i 个选，体积不超过 j 的最优方案的数量，所以最后 g[n,m]就是答案。

如果定义为==恰好==，g[0,i]，当 i>1 时，不可能什么都不选但体积不为0，所以只需初始化 g[0,0]=0。答案需要枚举 体积，累积价值最大的数量。

10 金明的预算方案

物品分为主件和附件，选择附件的前提是已经选择对应的主件，这是有限制的分组背包问题。

枚举主件物品，设 h 数组为临时数组，临时数组把主件选上，然后将附件视为 01 背包问题，枚举，然后将 h 数组的最大值保存到 f 数组即可。

for(int i=0; i<z.size(); i++) // 遍历主件
{
    int v1=z[i].v,p1=z[i].p,id=z[i].id;
    for(int j=v1; j<=n; j++)  // 选上主件 初始化 f 
        h[j]=f[j-v1]+p1*v1;
    for(int j=0; j<fu[id].size(); j++) // 遍历附件，已满足前提
    {
        int v2=fu[id][j].v, p2=fu[id][j].p;
        for(int k=n; k>=v1+v2; k--)  // 01背包
            h[k]=max(h[k],h[k-v2]+v2*p2);
    }
    for(int j=v1; j<=n; j++)  f[j]=max(f[j],h[j]); 
}

11 有依赖的背包问题

对于父结点 x，将它体积作为分组对象，分为 0，1，···，m-v[x]。每次只能选择一个体积去做 分组 背包。

void dfs(int x,int mx) // mx 表示父结点 x 以及子结点可用的体积
{
    for(int i=h[x];i!=-1;i=ne[i])
    {
        int y=e[i];
        dfs(y,mx-v[x]);
		for(int j=mx-v[x];j>=0;j--)  // 先给 x 腾出来空
            // 一次只能选取一个体积,一种体积代表一种最大组合
			for(int k=0;k<=j;k++) 
    			f[x][j]=max(f[x][j],f[x][j-k]+f[y][k]);
    }
    // 最后考虑根节点
    for(int j=mx;j>=v[x];j--)   // 选上 x 
        f[x][j]=f[x][j-v[x]]+w[x];
    for(int j=0;j<v[x];j++)   f[x][j]=0;  // 选不下，因此要置 0
}
dfs(root,m);

void dfs(int x,int mx)
{
    // 先把根节点放进去
    for(int j=mx;j>=v[x];j--) f[x][j]=f[x][j-v[x]]+w[x];
    for(int i=h[x];i!=-1;i=ne[i])
    {
        int y=e[i];
        dfs(y,mx);
        for(int j=mx;j>=v[x];j--) // 开始分组背包
            for(int k=0;k<=j-v[x];k++)  // 枚举每一种体积
                f[x][j]=max(f[x][j],f[x][j-k]+f[y][k]);
    }
}
